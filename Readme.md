# Linda

## Quick start

### Ubuntu

```sh
sudo apt-get update
sudo apt-get install cmake ninja-build
make prepare build run
```

### Other systems

1. [Install Cmake](https://cgold.readthedocs.io/en/latest/first-step/installation.html)

2. [Install Ninja](https://ninja-build.org)

3. Run Make command

    ```sh
    make prepare build run
    ```
